# microcrawler

## Getting Started

### Clone this repository

```
$ git clone https://github.com/ApolloCrawler/microcrawler.git
```

### Enter directory with sources

```
$ cd microcrawler
```

### Init submodules

```
$ git pull && git submodule init && git submodule update && git submodule status
```
